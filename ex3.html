    • My submission for exercise 2 is a website that consumes web services. The main focus of this project is to inform visitors about the 'POKEMON' franchise. The "Pokemon Stats" tab is for generating pokemon stats based on a pokemon's number while the "Relevant Videos" tab links the user to a popular youtube video on the subject.

    Enter an integer value and press the pokemon button to determine a pokemon by its number!

    Press this button to generate a random POKEMON youtube video!
